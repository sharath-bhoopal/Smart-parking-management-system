# Smart Parking Management System

## Overview
A Smart Parking Management System developed using the ServiceNow platform to simplify parking slot booking and management.

## Features
- User Registration
- Parking Slot Booking
- View Available Parking Slots
- Booking Cancellation
- Admin Dashboard
- Automated Notifications

## Technologies Used
- ServiceNow
- Flow Designer
- Service Catalog
- Business Rules
- Client Scripts
- UI Policies
- Tables
